# Retailer List

### Built by React Native


## Getting Started

First, run the development expo server:

```bash
npm start
# or
yarn start
# or
pnpm start
```

Second, press following keys to open the app in simulators to see the result.

```bash
› Press a │ open Android
› Press i │ open iOS simulator
› Press w │ open web

# or

npm / yarn android
# or
npm / yarn ios
# or
npm / yarn web


```
